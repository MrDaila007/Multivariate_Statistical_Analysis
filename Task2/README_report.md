# Инструкция по выполнению Task2: Анализ главных компонент (PCA)

## Задание

1. Найти данные с минимум 5 переменными
2. Построить главные компоненты для случаев с 2 и 3 компонентами
3. Интерпретировать результаты (что означает PC1, PC2)
4. Подготовить PDF отчет

## Что включено

### 1. R код (`task2.1.r`)

Код включает:
- **Генерацию многомерных данных** (6 переменных, n=200)
- **PCA анализ с 2 главными компонентами**
- **PCA анализ с 3 главными компонентами**
- **Визуализацию результатов:**
  - Scree plots (графики собственных значений)
  - Biplots (проекции данных и переменных)
  - Корреляционная матрица
  - Графики нагрузок (loadings)
- **Сохранение результатов** в текстовый файл

### 2. LaTeX отчет (`report.tex`)

Отчет включает:
- Теоретическую часть о методе PCA
- Описание данных
- Результаты анализа для 2 и 3 компонент
- Интерпретацию PC1, PC2, PC3
- Визуализацию результатов
- Выводы

## Как использовать

### Шаг 1: Установка пакетов R

```r
install.packages(c("MASS", "ggplot2", "corrplot", "factoextra", "FactoMineR"))
```

### Шаг 2: Запуск R кода

```r
source("Task2/task2.1.r")
```

Это создаст:
- `Task2/Task2_Results.pdf` - scree plots и графики дисперсии
- `Task2/Task2_Biplot.pdf` - биплоты
- `Task2/Task2_Correlation.pdf` - корреляционная матрица
- `Task2/Task2_Loadings.pdf` - графики нагрузок
- `Task2/Task2_analysis_results.txt` - численные результаты

### Шаг 3: Компиляция LaTeX отчета

**Используйте XeLaTeX** (рекомендуется для русского языка):

```r
# В R:
tinytex::xelatex("Task2/report.tex")
tinytex::xelatex("Task2/report.tex")  # Дважды для правильных ссылок
```

Или используйте скрипт:
```r
source("Task2/compile_report.R")
```

**Важно:** Убедитесь, что все PDF файлы с графиками созданы перед компиляцией LaTeX!

## Структура отчета

1. **Теоретическая часть** - основы метода PCA
2. **Описание данных** - 6 переменных с корреляционной структурой
3. **Результаты анализа** - таблицы собственных значений и нагрузок
4. **Интерпретация PC1** - что означает первая главная компонента
5. **Интерпретация PC2** - что означает вторая главная компонента

7. **Визуализация** - графики результатов
8. **Выводы** - обобщение результатов

## Описание данных

Синтетический набор данных с 6 переменными:
- **Экономические показатели** (3 переменные): GDP_Growth, Industrial_Output, Trade_Balance
- **Социальные показатели** (2 переменные): Education_Level, Healthcare_Index
- **Случайный фактор** (1 переменная): Random_Factor

Переменные имеют корреляционную структуру:
- Экономические переменные коррелированы между собой (ρ ≈ 0.6-0.7)
- Социальные переменные коррелированы (ρ ≈ 0.75)
- Случайный фактор независим от остальных

## Примечания

- Код использует `set.seed(2025)` для воспроизводимости результатов
- Данные стандартизируются перед PCA анализом
- Используется корреляционная матрица (не ковариационная)
- Все графики сохраняются в PDF формате для включения в отчет
